## Deployment Options

FlyingDuck Agent  DuckDefender can be deployed in two ways, by adding the duckdefender agent file in your respective work flow and other is using On-premise Runner.

- **Workflows**:\
The code will be scanned by the runner created by the version control system.

- **Runner**:\
In the workflows, build minutes are limited. If we exceed the allotted build minutes, we cannot run the runner. This is where the on-premise runner comes into play.
